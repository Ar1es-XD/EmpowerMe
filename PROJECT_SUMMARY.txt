==================================================

  ✓ Last Name: non-empty, min 2 chars
  ✓ Email: valid email format
  ✓ Phone: 10+ digits
  ✓ State: required selection
  ✓ Password: 8+ characters
  ✓ Confirm: must match password
  ✓ Terms: must be checked

Login:
  ✓ Email: valid email format
  ✓ Password: non-empty
  ✓ Verify against stored data
  ✓ Email must exist
  ✓ Password must match
