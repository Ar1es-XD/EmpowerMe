## What's in localStorage?

When you signup, your data is saved in browser storage:
- First Name, Last Name, Email
- Phone, State, Password
- Account creation timestamp

**To clear data:**
- Open DevTools (F12)
- Application → Local Storage → Clear All
- Or click "Sign Up" to create a new account

## Troubleshooting

**Page not loading?**
- Make sure `empowerme.html` and `empowerme.js` are in same folder
- Try clearing browser cache (Ctrl+Shift+Delete)
- Use Chrome, Firefox, Safari, or Edge

**Login not working?**
- Make sure you signed up first
- Check email and password match exactly
- Clear localStorage and signup again

**JavaScript errors?**
- Open DevTools (F12)
- Check Console for errors
- Verify both HTML and JS files are present

## Browser DevTools Tips
